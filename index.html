We Are Blob.


A Fully Preset Multipurpose/Moderation Bot Meant For All Types Of Servers.


Created By Tropii, Blob is a friendly bot that uses a standardized prefix rather than slash commands, as meant to be a rather more simplistic bot to use on discord.

Some of our commands start with "-", and include snipe, dictionary, and an improved help command if you're confused.

Frequently asked questions

What Is Blob?
Blob is a multi moderation tool and as well an entertainment discord bot with different commands made to keep your server a perfect place.
How can I find
all the available
commands?
To check all the commands simply type -help and all the options with explanations how to use them will be provided by the bot.
What do I do if I
have any issues or
found glitches in
the bot?
We have created a ticket system to report any technical issues with the bot simply type -ticket in the support server and we will provide assistance as quick as possible.
How can I add Blob
to my server?
Just press the invite button here!
What makes Blob
special?
By combining all the moderation and entertainment bots avaliable, we are developing and improving the bot as much as possible so it can eventually serve as a multipurpose bot that can provide you all your needs without several other bots on your server!
How can I make
suggestions or tips
for the bot?
We are always down for any advice, tips or suggestions. So in light of that, be sure to open a ticket in the support server and let us know. We will see what we can manage to add or fix!
How can I join
the staff or
developer team?
We will host applications every now and then when we are in need of new staff. If you are an experienced bot developer, or you know something that can possibly help us, create a ticket in our support server!

Lastly, As A Guarantee, We Do Not Store Any Personal Info Or Data At All!
